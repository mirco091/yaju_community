# YAJU VAULT

YAJU VAULT is a Web3-style community oripa marketplace concept.
Users can open community-made boxes with YAJU Coin, list prizes, share revenue, and build holder-driven gacha economies.

> This repository is a front-end MVP / concept prototype. Payment, token, custody, shipping, legal compliance, and randomness verification must be implemented carefully before production.

## Concept

**Holder-powered next-generation oripa vault.**

- Open boxes with YAJU Coin or supported altcoins
- List cards / NFTs / digital rewards as community prizes
- Revenue share between creators and platform
- Platform earns fees from gacha opens, listings, marketplace trades, and premium effects
- Rankings, live openings, and community boxes drive viral loops

## Core Features

- Premium dark casino-style landing page
- Community box cards
- Live gacha feed
- Seller ranking
- Listing CTA
- YAJU Coin balance mock UI
- Mobile-first app-style layout

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
