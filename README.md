# YAJU VAULT

Community-powered Web3 Oripa marketplace demo UI.

## Concept
- Users can open holder-made Oripa boxes with YAJU Coin
- NFT/Coin holders can list prizes and create boxes
- Platform revenue comes from fees
- Designed as a dark luxury Web3 casino/oripa style UI

## Run
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- lucide-react

## Pages included
- Hero / wallet connect mock
- Live gacha panel
- Popular holder boxes
- Listing form mock
- Live feed
